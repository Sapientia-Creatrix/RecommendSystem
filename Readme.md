# LearningCourseRecommender 使用範例
這隻程式的目的是讓 express 後端呼叫，並回傳推薦的課程詳細資訊
本程式需傳入三個參數
user id: 用於識別唯一使用者
recommendWay: 用於決定要使用哪一種推薦方法，可以為 "recommend"
sortWay: 用於決定要使用哪一種排序方法決定結果的順序，可以為 "rating"

## 執行
```
python -u "e:\progremmingFile\Github\Sapientia-Creatrix\RecommendSystem\LearningCourseRecommender.py" 1 recommend rating
```
